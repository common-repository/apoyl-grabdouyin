== Description ==

通过抖音分享视频链接，一键采集抖音视频到自己网站上，非常方便实用的工具。

## 插件功能

* 支持编辑器下面输入抖音链接，一键采集抖音视频（一键抓取抖音视频）到网站上，并快捷发布，提供使用者效率
* 支持视频本地化，这样及时抖音平台关闭分享，网站上也可以继续播放视频

此插件特此申明：方便用户抓取合规及原创作者授权抖音视频，请勿把文章用于商业及非法用途。
